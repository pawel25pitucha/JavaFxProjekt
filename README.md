# JavaProjekt

**Teamsport matches organizer.
Window JavaFx application with simple SQL datatbase.**


**Program contains:**\
-CRUD to each table in database (10).\
-MD5 alghorithm to make your password save.\
-User friendly GUI.\
-Search panel with filters.\
and more...
